# BLOOM

This is my first time creating a keyboard. BLOOM is an 80% mechanical keybooard
It also has a gradient keycap colour.

# BOM
- 9x MX_PCB_1.00u Keyswitches
- 9x LED_SK6812MINI
- 1x XIAO-RP2040-DIP 
- 9x Diode_DO-35
- 1x RotaryEncoder_Alps_EC11E-Switch_Vertical_H20mm
- 1x OLED_128x32
